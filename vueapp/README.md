# Challenge Frontend #

## Instalación ##
Instrucciones:  
1.- Situarse en la raíz de esta carpeta vueapp  
2.- Ejecutar comando npm install  
3.- Ejecutar comando npm run serve  

```
$ git clone https://github.com/raulidavid/challenge.git
```

```
$ cd challenge/vueapp
```

```
$ npm install
```

```
$ npm run serve
```

```
$ Abrir navegador http://localhost:8080/
```
