## Installation ##
Instrucciones:
1.- Situarse en la raíz de este carpeta LARAVEL-PROJECT
2.- Ejecutar comando composer install
3.- Ejecutar comando php artisan serve

```
$ composer install
```

```
$ php artisan serve
```
